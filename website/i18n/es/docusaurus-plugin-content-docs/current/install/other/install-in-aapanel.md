# Instalación en aaPanel

## Introducción

`aaPanel` es un panel de control de alojamiento web potente y fácil de usar para servidores Linux. Permite la instalación con un clic de entornos de desarrollo LNMP/LAMP/OpenLiteSpeed y software.

[Haz clic aquí para implementar aaPanel en tu servidor](https://www.aapanel.com/new/download.html?r=dk_tianji)

> Aplicable solo para la versión 7.0.11 de aaPanel y superiores.

## Lista de Actualización y Búsqueda

Barra de navegación `docker` -> `instalación con un clic` -> `buscar tianji`

![](/img/docs/aapanel/1.png)

> Si no puedes encontrarlo, por favor haz clic en el botón `Actualizar lista de aplicaciones`.

## Completa el formulario para instalar

![](/img/docs/aapanel/2.png)

Puedes completar el formulario rápidamente para instalar.

Solo necesitas esperar un momento. Luego podrás abrir tu propio Tianji en tu navegador.

> URL: `http://<tu-ip>:12345`  
> Nombre de usuario predeterminado: `admin`  
> Contraseña predeterminada: `admin`
