# Telemetría

## Acerca de la Telemetría Anónima  

Para mejorar la experiencia del usuario, hemos **habilitado por defecto la telemetría anónima** en su servicio Tianji autoalojado.  

Entendemos que la privacidad de los datos es importante para usted. Tenga la seguridad de que esta telemetría tiene las siguientes características:  

- **Completamente Anónima**: No recopilamos ninguna información personal identificable ni contenido del servicio.  
- **Alcance Extremadamente Limitado**: Solo incluye datos básicos de uso, lo que nos ayuda a comprender mínimamente el uso de Tianji para mejorar continuamente el producto.  
- **Totalmente Transparente**: Todas las transmisiones de datos de telemetría pueden observarse en el panel de red de su servicio, asegurando completa transparencia.  

Además, respetamos completamente su elección. Si prefiere **desactivar por completo la telemetría anónima**, simplemente configure la siguiente variable de entorno:  

```bash
DISABLE_ANONYMOUS_TELEMETRY=1
```

Para obtener instrucciones de configuración detalladas, consulte la Documentación de Variables de Entorno.

Con este enfoque, queremos mejorar Tianji mientras garantizamos la privacidad y seguridad. Si tiene alguna pregunta o comentario, ¡no dude en contactarnos!
